## What this repo is

This is a personal learning repo created by [MeetaDev's Momentum](https://meetadev.com) system. Every assignment is a real coding task reviewed by Alex, an AI senior engineer who reads your actual code, posts inline comments, and tracks your growth across concepts.

**Goal:** html css fundamentals

## How the workflow works

1. **Alex assigns** — each day, Alex prepares a coding assignment matched to your current phase and skill level
2. **You code on a feature branch** — write your solution on the assignment branch
3. **Open a PR** — create a pull request from your branch to `main` on GitHub
4. **Alex reviews your code** — expect specific, line-by-line feedback on what works and what to improve
5. **Address feedback or celebrate approval** — push fixes if changes are requested, or your PR gets merged when approved
6. **Your work becomes part of main** — every merged PR builds your portfolio

## Branch naming convention

```
assignment/week-N-day-N-concept-name
```

Example: `assignment/week-1-day-2-async-functions`

## After review

If Alex requests changes: push your fixes to the same branch and Alex will review again.

If Alex approves: your PR is merged into main automatically. On to the next assignment.

## Your learning journey

Each merged PR is a unit of verified competence. Your git history is your portfolio — every commit tells the story of what you learned and how you grew.
